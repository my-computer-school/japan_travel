# japan_travel
일본 여행

## 내질문 : 11월13일 인천공항에서 07시 출발해 간사이공항 도착, 11월18일 나리타공항 11시 출발 인천공항 도착, 총 4인 가족이고 총 예산은 4백만원이야.

일본 여행을 위한 일정과 예산을 고려하여 계획을 세워보겠습니다. 아래는 4인 가족을 위한 5일간의 일본 여행 일정입니다. 예산은 4백만 원으로 설정하였습니다.

### 여행 일정

#### 1일차: 11월 13일 (화요일) - 인천 → 간사이
- **07:00**: 인천공항 출발
- **09:30**: 간사이공항 도착
- **오후**: 오사카 시내로 이동 (공항 리무진 버스 또는 기차 이용)
- **점심**: 오사카에서 유명한 타코야키 또는 오코노미야키
- **오후**: 오사카 성 방문
- **저녁**: 도톤보리에서 저녁 식사 (길거리 음식 탐방)
- **숙소**: 오사카 시내 호텔 체크인

#### 2일차: 11월 14일 (수요일) - 오사카
- **오전**: 유니버설 스튜디오 재팬 방문 (사전 예약 필수)
- **점심**: 유니버설 스튜디오 내 식사
- **오후**: 유니버설 스튜디오에서 즐기기
- **저녁**: 신사이바시에서 저녁 식사
- **숙소**: 오사카 시내 호텔

#### 3일차: 11월 15일 (목요일) - 교토
- **오전**: 교토로 이동 (기차 이용)
- **오전**: 기온 거리 탐방 (전통 찻집, 기모노 체험)
- **점심**: 교토의 전통 음식 (유도후 등)
- **오후**: 아라시야마 대나무 숲 방문
- **저녁**: 교토에서 카이세키 요리 체험
- **숙소**: 교토 시내 호텔 체크인

#### 4일차: 11월 16일 (금요일) - 교토 → 도쿄
- **오전**: 금각사 방문
- **점심**: 교토에서 점심 식사
- **오후**: 신칸센을 이용해 도쿄로 이동
- **저녁**: 도쿄 시부야 탐방 (시부야 스크램블 교차로)
- **숙소**: 도쿄 시내 호텔 체크인

#### 5일차: 11월 17일 (토요일) - 도쿄
- **오전**: 아사쿠사 방문 (센소지 사원)
- **점심**: 아사쿠사에서 점심 식사
- **오후**: 하라주쿠 탐방 (타케시타 거리)
- **저녁**: 신주쿠에서 저녁 식사
- **숙소**: 도쿄 시내 호텔

#### 6일차: 11월 18일 (일요일) - 도쿄 → 인천
- **오전**: 체크아웃 후 공항으로 이동
- **11:00**: 나리타공항 출발
- **13:30**: 인천공항 도착

- **항공권**: 약 120만 원 (4인 기준, 왕복)
- **숙소**: 약 120만 원 (5박, 4인 기준)
